# Java Practice Programs

This repository contains Java programs covering core programming concepts and problem solving.

- OOP concepts (Encapsulation, Inheritance, Polymorphism)
- Collections (List, Set, Map)

- String manipulation
- Logical problem solving

## 🛠️ Tech Stack
- Java

## ▶️ How to Run
1. Clone the repository
2. Open in IDE (Eclipse / IntelliJ)
3. Run individual classes

To strengthen core Java concepts required for automation testing and backend logic.
